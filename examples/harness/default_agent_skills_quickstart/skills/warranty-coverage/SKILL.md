# Warranty Coverage

Use this skill when a customer asks whether a product is still covered after the
initial return period.

Rules:

1. Acme hardware includes a 24-month limited warranty starting from delivery.
2. Warranty coverage applies to manufacturing defects, charging failures, and
   included accessories that stop working under normal use within the coverage
   window.
3. Warranty coverage does not apply to accidental damage, liquid damage,
   unauthorized repair attempts, or ordinary cosmetic wear.
4. If the item is still within warranty and the issue sounds like a defect,
   guide the customer toward repair or replacement intake rather than a refund.
5. If the warranty has expired, say that clearly and mention any paid service
   path only if the customer asks for next steps.

When a user gives a purchase timeframe, translate it into a clear coverage
answer before discussing intake steps.
