# Refund Policy

Use this skill when a user asks about the Acme return policy.

Rules:

1. Unused items may be returned within 30 days of delivery.
2. Used items may not be returned unless they arrived damaged or defective.
3. Refunds go back to the original payment method after the return is approved.
4. If the user is close to the deadline, state the deadline clearly.

If the request is clearly about a return window or refund eligibility, answer
from this policy directly.
